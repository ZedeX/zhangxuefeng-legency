# 张雪峰教育观点问答

基于 100+ 篇张雪峰公开访谈整理，打造一个纯静态离线可用的问答网站，解答学生关于考研、择校、高考志愿填报、专业选择等困惑。

## 在线访问

直接访问 (https://ZedeX.github.io/zhangxuefeng-legency/)

## 项目特点

- **纯静态网页**：所有数据内联，无需后端服务器，双击 `index.html` 直接打开使用
- **完整离线可用**：不依赖任何外部资源，完全本地运行
- **支持关键词搜索**：松散匹配+得分排序，找到你最关心的问题
- **分类导航**：按高考志愿、专业选择、考研、人生建议分类浏览
- **热门问题快捷入口**：点击直接查看常见问题
- **URL 锚点状态保持**：刷新、浏览器前进后退都能保持当前分类状态
- **移动端响应式**：支持手机、平板访问
- **安全可靠**：完整 XSS 防护，所有内容都经过 HTML 转义

## 技术说明

- 纯 HTML + CSS + JavaScript，无框架依赖
- 关键词匹配得分算法：问题标题匹配（3分）+ 标签匹配（2分）+ 回答内容匹配（1分）
- 停用词过滤：过滤掉"我"、"请问"、"什么"等无意义疑问词，提高匹配准确率
